# 📘 Module 1: Python Basics

Welcome to the **first module** of your Python journey! This module is designed for **absolute beginners** with zero programming experience. By the end of this module, you will have a solid foundation in Python fundamentals.

---

## 📌 Overview of This Module

This module covers the **building blocks** of Python programming. You will learn how Python works, how to store and manipulate data, how to interact with users, and how to perform operations on data.

Think of this module as learning the **alphabet and grammar** of Python — without this, you cannot write meaningful programs.

---

## 📚 What You Will Learn

After completing this module, you will be able to:

| Skill | Description |
|-------|-------------|
| ✅ Understand Python | Know what Python is and why it's popular |
| ✅ Write first program | Create and run your first Python script |
| ✅ Use variables | Store and retrieve data in memory |
| ✅ Work with data types | Understand integers, floats, strings, booleans |
| ✅ Handle input/output | Take user input and display output |
| ✅ Convert data types | Change one type to another (type casting) |
| ✅ Use operators | Perform arithmetic, comparison, and logical operations |

## 💡 Tips Before Starting

### For Absolute Beginners:

1. **Don't just read — TYPE the code yourself**
   - Typing helps you remember better than copy-pasting
   
2. **Make mistakes intentionally**
   - Try changing code to see what breaks
   - This teaches you debugging skills
   
3. **Practice the exercises at the end of each file**
   - Reading without practice = forgetting

### For Those with Some Experience:

1. **Don't skip the basics**
   - You might discover something new
   
2. **Focus on the "Interview Insight" sections**
   - These highlight commonly asked questions
   
3. **Try the advanced code examples**
   - Challenge yourself

## 📝 Quick Reference Card

```python
# Print output
print("Hello, World!")

# Variables
name = "Python"
age = 30
price = 99.99
is_awesome = True

# Input
user_input = input("Enter your name: ")

# Type casting
num_str = "42"
num_int = int(num_str)

# Operators
result = 10 + 5      # Addition: 15
result = 10 - 5      # Subtraction: 5
result = 10 * 5      # Multiplication: 50
result = 10 / 5      # Division: 2.0
result = 10 // 3     # Floor Division: 3
result = 10 % 3      # Modulus: 1
result = 10 ** 2     # Power: 100
```
